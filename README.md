# Password-Manager-App
Password Manager App to save all your passwords.
# Description
This password manager is a great tool to generate strong passwords and save them. First, enter the name of the website for which you want to store your password, along with
the email address or username associated with that website. On clicking the generate password button, a complex password is automatically generated, containing various
combinations of letters, symbols and numbers. As soon as the password is generated, it is copied to the clipboard. Alternatively, the user can write their own password.
